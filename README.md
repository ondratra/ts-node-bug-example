# Showcase of bug

Issue can be tracked [here](https://github.com/TypeStrong/ts-node/issues/573)

# Install
You will need nodejs and npm/yarn.

# Run
```
yarn run build
# this will cause error "'Promise' only refers to a type, but is being used as a value here."
```
